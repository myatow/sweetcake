## Установка и запуск
1. Клонируйте репозиторий:
   git clone https://github.com/ВАШ_АККАУНТ/sweetcake.git ;
   cd sweetcake
2. Установите зависимости:
npm install
3. Создайте файл .env в корне проекта:
TELEGRAM_TOKEN=ваш_токен_бота ;
CHAT_ID=ваш_chat_id
4. Запустите сервер:
node server.js
5. Откройте локальный сервер: http://localhost:3000
